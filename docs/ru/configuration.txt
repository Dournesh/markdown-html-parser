title - название опции.
Список всех title: headers, font, line-break, li, list, link, image, code, blockquote.

===========================

КОНФИГУРАЦИЯ ДЛЯ ЗАГОЛОВКОВ.

===========================

---------------------------------------------------------------------------------------------------
Включить парсинг заголовков только 3 уровня (###), и html элементу на выходе присвоить класс level3

{
    title: 'headers',
    included: true, // Включить парсинг заголовков. false - выключить парсинг заголовков.
    level: {
        number: 3,
        className: 'level3'
    }
}

---------------------------------------------------------------------------------------------------

--------------------------------------------------------------------------------------------------------------
Включить парсинг заголовков 1, 2, 5 уровней, и элементам на выходе присвоить классы, соответствующие className

{
    title: 'headers',
    included: true,
    levels: [{
        number: 1,
        className: 'level1'
    }, {
        number: 2,
        className: 'level2'
    }, {
        number: 5,
        className: 'level5'
    }]
}

--------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------
Включить парсинг всех уровней заголовков, и каждому уровню заголовка (h1-h6) присвоить название класса

{
    title: 'headers',
    included: true,
    levels: "all",
    classNames: {
        h1: String,
        h2: String,
        ...
        h6: String
    }
}

------------------------------------------------------------------------------------------------------

========================

КОНФИГУРАЦИЯ ДЛЯ ШРИФТОВ.

========================

------------------------------------------------------------
Включить парсинг жирного шрифта. Разрешен только символ "*".
Пример: ** bold **, **bold** (НЕ __ bold __, __bold__).

{
    title: 'font',
    included: true,
    font: {
        title: 'bold',
        included: true,
        allowed: '*'('_', 'all'),
        className: 'bold'
    }
}

------------------------------------------------------------

-------------------------------------------------------------------------------------------------
Включить парсинг всех шрифтов. У жирного шрифта разрешены любые символы (**, __).
У наклонного шрифта разрешен только символ "_" (__).
Т.к. символ перечеркнутого текста только один (~), то поэтому allowed не включен.

{
    title: 'font',
    included: true,
    fonts: [{
        title: 'bold',
        included: true,
        allowed: 'all',
        className: 'bold'
    }, {
        title: 'inclined',
        included: true,
        allowed: '_',
        className: 'inclined'
    }, {
        title: 'strike',
        included: true,
        className: 'strike'
    }]
}

-------------------------------------------------------------------------------------------------

------------------------------------------------------------------------
Включить парсинг всех шрифтов со всеми разрешенными символами.
Так же указываются имена классов каждого шрифта (bold, inclined, strike)

{
    title: 'headers',
    included: true,
    fonts: 'all',
    "classNames": {
      "bold": "bold",
      "inclined": "inclined",
      "strike": "strike"
    }
}

------------------------------------------------------------------------

КОНФИГУРАЦИЯ ДЛЯ ГОРИЗОНТАЛЬНОЙ ЛИНИИ.

======================================

-----------------------------------------------------------------------------
Включить парсинг горизонтальной линии, разрешенный символ - "*" (**********).

{
    title: 'line-break',
    included: true,
    allowed: "*"(-),
    className: "hr"
}

-----------------------------------------------------------------------------

------------------------------------------------------------------------
Включить парсинг горизонтальной линии, разрешены все символы ("*", "-").

{
    title: 'line-break',
    included: true,
    allowed: "all",
    className: "hr"
}
